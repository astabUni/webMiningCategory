# webMiningCategory

This project extracts programming languages categorized by their paradigm based on categories from dbpedia via SPARQL queries and visualizes the retrieved data into a graph using graphviz.


# Setup

## Prerequisites
* Python 3.7
* graphviz (https://pypi.org/project/graphviz/)
* SPARQLWrapper (https://rdflib.github.io/sparqlwrapper/)

## How to Run

* Go to the webMiningCategory folder
```
$ python mining.py
```
